# Description
Simple rock-paper-scissors game

# Install
```
apt-get install git python
git clone https://github.com/gl1tchdev/RPSmin
cd RPSmin
python -m venv env
source env/bin/activate
pip install -r requirements.txt
pip install gunicorn
```
# Run
```
python app.py
```
